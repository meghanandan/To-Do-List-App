# To-Do List App

A simple To-Do List application built with React.js and Bootstrap.

## Features

- Add tasks
- Mark tasks as complete
- Delete tasks
- Responsive design

## Getting Started

Follow these instructions to set up and run the project on your local machine.

### Prerequisites

- Node.js (v18.18.2)
- npm (v10.5.0)
